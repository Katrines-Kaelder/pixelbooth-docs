# Pixelbooth Docs

This repository hosts the public documentation site for Pixelbooth:

https://katrines-kaelder.github.io/pixelbooth-docs/

Pixelbooth is a local-first photo booth system for Raspberry Pi based event
setups. The documentation covers local development, Raspberry Pi setup,
operator runbooks, architecture decisions, product notes, and the staged project
roadmap.

## Source Of Truth

This repository is a publishing target, not the documentation source.

The canonical documentation lives in the private `Katrines-Kaelder/pixelbooth`
repository under `.docs/`. Changes should be made there, reviewed there, and
then published here by the docs workflow.

## Publishing Flow

The private source repository builds the Docusaurus site and publishes the
generated static output into this repository.

- Pull requests in the source repository run a docs build check.
- Pushes to the source repository's `docs` branch publish the site.
- The `Docs` workflow can also be run manually when needed.
- This repository serves GitHub Pages from `main` and `/`.

## Repository Contents

Most files in this repository are generated by Docusaurus. Manual edits to the
generated site output will be overwritten by the next publish.

The only hand-maintained file expected to remain here is this `README.md`.
